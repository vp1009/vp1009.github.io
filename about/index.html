• 预防请求量过大时出现的高并发问题
• 使用nginx 反向代理实现高请求负载均衡

项目职责：
负责项目二开设计，开发支付模块，搭建分布式存储系统
技术要点：

•使用django rest_framework 编写 restful 风格 api
•基于 Docker 搭建 FastDfs，使用FastDfs分布式文件系统存漫画图片，增加静态文件的上传下载效率，增强体验感。
•使用视图集进行漫画调整，对漫画进行每日最新、最热、以及分页展示、方便用户选择以及漫画推送。
•使用递归对漫画进行处理，并Element-UI组件前端展示
•结合腾讯和阿里的开发平台实现三方支付模块
•搭建中间件，对权限进行检测，拦截恶意请求

项目三：分布式漫画抓取爬虫系统
项目简介：
该项目是一个爬取漫画素材及热度分析的项目，指让平台动态实时获取到同漫画平台的漫画热度&信息，并且及时更新漫画排行榜，提升市场人气，增加知名度，快速的抢占市场资源。
项目职责：
负责设计和开发基于 Scrapy 的分布式漫画抓取系统，优化爬虫策略    。
技术要点：

•使用 Scrapy_Redis 框架对漫画进行抓取
•利用 Redis 维护 Ip 代理池，提升漫画抓取的效率
•用 Xpath 对数据进行抽取和清洗
•利用 ECharts 对抓取后的数据进行可视化展示
•使用 Splash 针对异步动态数据加载进行反复测试和改进抓取方案
•使用Scrapy + Selenium 篡改中间件模拟真实请求防止检测
