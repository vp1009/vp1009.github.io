我们将颜色用变量控制

通过变量控制

# 声明变量
:root{
  --bg--color:'#fff';
  --a-color:'black';
}

使用变量

// 变量用法
color:var(--a-color);

写方法控制背景和字体颜色

1 需要个开关按钮

<h-switch v-model="colors" @change="change_back">白/黑</h-switch>

注：利用双向绑定给其默认值 

data(){
        return{
			colors:0, // 默认代表白色
        }
    }

2 创建change_back方法

change_back:function(){
		// 获取样式表
		var styles = getComputedStyle(document.documentElement) 
		if (this.colors != 0 ){
			// 给背景赋值
			document.documentElement.style.setProperty('--bg-color',"#292a2b")
			// 设置字体颜色
			document.documentElement.style.setProperty('--a-color',"white")
		}else{
			// 给背景赋值
			document.documentElement.style.setProperty('--bg-color',"#white")
			// 设置字体颜色
			document.documentElement.style.setProperty('--a-color',"#292a2b")
		}
	}

注：切换到暗黑主题时我们将字体改成白色

在指定时间运行代码

// 指定时间运行代码
		runtime:function(){;		
			var data = new Date()
