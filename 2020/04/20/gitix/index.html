什么是Git ?

2020-04-20 266 words
Git

GIT，全称是分布式版本控制系统，git通常在编程中会用到，并且git支持分布式部署，可以有效、高速的处理从很小到非常大的项目版本管理。分布式相比于集中式的最大区别在于开发者可以提交到本地，每个开发者通过克隆（git clone），在本地机器上拷贝一个完整的Git仓库。

git命令介绍

git clone 代码库的url
将线上库克隆到本地

git init
初始化git

git renote add origin (address) # 仓库地址
添加到远程仓库

git add -A
将工作区的修改提交到暂存区

git commit -m 'msg'
将暂存区的文件或目录提交到版本库区

git push origin master
将提交到本地库中的内容推送到远程库中


查看日志：git log
回滚（本地）：git reset --hard 版本id  
回滚（服务器）：git push HEAD --force  



git崩溃解决方法

    1):删除项目，全部重做

    2):删除.git，清空仓库，重新链接库，重新上传文件
